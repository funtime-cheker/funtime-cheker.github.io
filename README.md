# FunTime Checker — сайт проекта

Лендинг утилиты **FunTime Checker** — инструмента для проверяющих на сервере FunTime: мгновенный поиск следов чит-клиентов (MFT-индекс, Prefetch, журналы Windows, реестр).

## Стек

Чистый HTML / CSS / JavaScript. Без сборщиков, фреймворков и зависимостей — сайт работает из коробки на любом статическом хостинге.

```
index.html        — страница
assets/style.css  — стили и анимации
assets/app.js     — интерактив: живой поиск, терминал, вкладки, счётчики
.nojekyll         — служебный файл для GitHub Pages
```

## Как посмотреть локально

Открыть `index.html` в браузере — этого достаточно. Либо поднять локальный сервер:

```
python -m http.server 8080
```

и открыть http://localhost:8080.

## Деплой

Подходит любой статический хостинг: GitHub Pages, Cloudflare Pages, Netlify, Vercel или обычный nginx. Для GitHub Pages — залить содержимое папки в репозиторий и включить Pages в настройках (ветка `main`, папка `/`).

## Поддержка браузеров

Chrome, Edge, Firefox, Opera, Safari — актуальные версии. Адаптивная вёрстка от 375px. Учитывается `prefers-reduced-motion`.
